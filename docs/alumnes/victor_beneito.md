# Victor Beneito LLuch

## Presentació
Soc alumne del curs de 2ón **Desenvolupament d'Aplicacions Webs (DAW)** en l'Institut l'Estació d'Ontinyent.

## Projecte
Página web d'una tenda online de venda de productes de llar amb el nom de *"El Hogar de tus Sueños"* amb productes com Estors Digitals, Estors LLiços, Fundes de sofá, Fundes Nordiques, Coixins, Mantels...

El frontend es farà en *React*, *Vite* i *Tailwindcss* i el backend en *Node.js*, *Express* i *Mongodb*

## Tecnologies a utilitzar

- React, Vite
- Taildwind, Css
- Node.js, Express i MongoDb
- GitHub
- AWS
